== Description ==

Performance Optimisation is a comprehensive WordPress plugin designed to dramatically improve your website's speed and performance. With an intuitive setup wizard and powerful optimization features, it helps you achieve faster load times, better SEO rankings, and improved user experience.

= Key Features =

**🚀 Page Caching**
* Advanced page caching system
* Smart cache invalidation
* Cache preloading for popular pages
* Mobile-specific caching

**⚡ File Optimization**
* CSS, JavaScript, and HTML minification
* File combination to reduce HTTP requests
* Critical CSS extraction and inlining
* JavaScript defer and async loading

**🖼️ Image Optimization**
* Automatic WebP and AVIF conversion

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "Performance Optimisation"
4. Click "Install Now" and then "Activate"
5. Follow the setup wizard to configure your optimizations

= Manual Installation =

1. Download the plugin zip file
2. Upload it to your `/wp-content/plugins/` directory
3. Extract the files
4. Activate the plugin through the WordPress admin
5. Run the setup wizard to get started

= After Installation =

1. Navigate to Performance Optimisation in your admin menu
2. Run the setup wizard for automatic configuration
3. Or manually configure settings in each tab
4. Monitor your performance improvements in the Analytics section

= Is this plugin safe to use? =

Yes! The plugin includes extensive safety measures:
* Automatic backups before major changes
* Rollback functionality if issues occur
* Extensive testing on various WordPress configurations

= Will this plugin slow down my admin area? =

No. The plugin is designed to optimize frontend performance without impacting admin functionality. All optimizations are applied only to public-facing pages.

= Can I use this with other caching plugins? =

While possible, it's recommended to use only one caching solution to avoid conflicts. This plugin provides comprehensive caching that typically replaces the need for other caching plugins.

== Advanced Configuration ==

= Developer Hooks =

The plugin provides numerous hooks for developers:

**Actions:**
* `wppo_before_optimization` - Fired before any optimization
* `wppo_after_optimization` - Fired after optimization complete
* `wppo_cache_cleared` - Fired when cache is cleared
* `wppo_image_optimized` - Fired after image optimization

**Filters:**
* `wppo_cache_exclusions` - Modify cache exclusion rules
* `wppo_minify_exclusions` - Exclude files from minification
* `wppo_optimization_settings` - Modify optimization settings
* `wppo_performance_thresholds` - Customize performance thresholds

= Configuration Constants =

Define these constants in wp-config.php for advanced control:

```php
// Disable specific features
define('WPPO_DISABLE_CACHING', true);
define('WPPO_DISABLE_MINIFICATION', true);
define('WPPO_DISABLE_IMAGE_OPTIMIZATION', true);

// Performance settings
define('WPPO_CACHE_EXPIRATION', 3600);
define('WPPO_MAX_IMAGE_SIZE', 2048);
define('WPPO_MINIFY_LEVEL', 2);

// Debug mode
define('WPPO_DEBUG', true);
```
